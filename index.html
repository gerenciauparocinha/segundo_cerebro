🚀 Como Usar

No Celular (Recomendado):

  1. Baixe o app Expo Go no seu celular
  2. Escaneie o QR Code do projeto
  3. Comece a usar imediatamente

No Computador:

  1. Clone o repositório
  2. Execute pnpm install && pnpm dev
  3. Acesse http://localhost:8081

🔐 Configuração de API

O app suporta múltiplos provedores de IA:

  • Groq: Rápido e econômico (recomendado)
  • OpenAI: GPT-4 e GPT-3.5
  • Anthropic: Claude

Configure sua API Key na tela de configuração do app.
